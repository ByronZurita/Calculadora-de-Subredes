# Calculadora de Subredes (.NET)

Una aplicación de consola simple para calcular subredes usando CIDR o VLSM.

## Funcionalidades

- Ingresar IP con CIDR (ej. `192.168.1.0/24`)
- Ingresar IP base y cantidad de hosts
- Soporte para VLSM (subredes con diferentes tamaños)
- Guarda resultados en un archivo `.txt`

## Basado en

* [vlsmcalc.vercel.app](https://vlsmcalc.vercel.app)
* [michqo/vlsmcalc](https://github.com/michqo/vlsmcalc)
